回首


        

        

至少有一年零三个月，我没有听过这首歌了。那时，晚上十点半，从 C 楼出来，我夹着 Pad 往回走，憨憨式用它充当播放器。那时，闪电接口的 EarPods 还刚刚服役不久，在退烧之路上扮演着重要作用。那时，我还可以很大程度的专注在自习上，不曾有来自某些方面的焦虑。


要怎么去形容Memories...Do Not Open这张专辑呢？我想，从Something Just Like This起步，到颓废The One，再到完美 bridge Break Up Every Night，之后到三短两长部分的




We were staying in Paris
To get away from your parents




超越现实感加




If we go down then we go down together
We’ll get away with everything
Let’s show them we are better




的数模美赛即视感Paris，标准烟鬼流行乐Honest、Young，再到饱含冲击力、尽是不舍的最后一曲Last Day Alive：这趟旅程，是20岁的放纵、迷茫、逃避、上进、压抑的自我写照。


明日
虽然不难看出的是，我更珍重Paris，但今日的主题很确定的，是Break Up Every Night。原因是明天，我会去当面尝试再和某些最重要的人做post-break-up negotiation：




So I build bridges up, the bridges up
Build the bridges up again
Build bridges up, the bridges up
Build the bridges up again




如果能和曲中的渐强一样最终过渡回到老chorus，那会是非常好的结果了。
