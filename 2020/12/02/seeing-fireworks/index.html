易逝，花火

  1. 2017
  2. 2018
  3. 2019
  4. 2020#1
  5. 2020#2

2017

你带着我试音，有两首歌我不会忘，一首是和A35关系很大的このiは虚数，另一首就是打上花火。

显而易见，这首歌瞬态很突出，在听感塑造上起了很大帮助。结构上的安排稳定、合理，长度也包含满满诚意。

我看到了你的1060Ti，它竟吃力地带起了4K 60fps的PUBG；我也看到了你的电饭煲，它藏在书桌柜子里默默地煲着粥；我当然也体验了你的光驱，放了一把吃灰的Wild World 1CD。

学业如何？还准备烧多久？

2018

我几乎要对你彻底失望了。意外地，你带上GR09把这首歌演绎的精致、动情。我知道有电平在作祟，但我愿吧所有功劳都归给你。原来你一直无法替代。

送你们俩走的时候，我的内心虽算平静，但我知道，一个时代结束了，这个时代已经走过了它所谓的全盛巅峰，这个时代已无法支撑。

你们被温柔对待了吗？还是只是一再经历别离？

2019

我已经蜷缩到只敢听翻唱了。罗马音战士、手机麦战神，你们尽力了，开声闷也没关系，效果已经很惊艳了。

2020#1

陌生、虚实交错的环境里，你在没有任何事先预料的情况下，出现了。还算安静，还算自然，还算自如，我侧过头去，尽情享受你在睽别一年后泼洒在我的整个身体上。

陌生的他们是否有所察觉？陌生的他们是否已遗忘这一切？

2020#2

轰鸣的电机停转了，作业也不出意外地暂时停摆了。环境在撺掇，我算是接受了邀请。

24bit、48kHz的格式竟显得如此陌生。同样是有陌生的他们。不同的是，今天你的朋友是KXXS和rMBP。

我有点紧张，当然是在担心听感退化。

你没变，原来没变。我变了，当然变了。我和你，都变了，也都没变。那一时刻的花火总会消逝，但永久定格。
